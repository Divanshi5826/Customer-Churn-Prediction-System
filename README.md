# Customer Churn Prediction System

## Overview

Customer Churn Prediction System is a Machine Learning project designed to predict whether a customer is likely to discontinue a service based on demographic, subscription, and behavioral data. The objective is to help organizations identify at-risk customers and support customer retention strategies through predictive analytics.

---

## Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Customer Churn Prediction
* Model Evaluation and Performance Analysis
* Data Visualization and Reporting

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

---

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Customer Churn Prediction

---

## Models Used

* Logistic Regression
* Random Forest Classifier

---

## Dataset

The project utilizes the Telco Customer Churn Dataset containing customer demographic information, subscription details, account information, and service usage patterns.

---

## Key Outcomes

* Identified important factors influencing customer churn behavior.
* Applied feature engineering and preprocessing techniques to improve data quality.
* Trained and evaluated machine learning classification models for churn prediction.
* Compared model performance using standard evaluation metrics.
* Generated visual insights to support data-driven business decisions and customer retention strategies.
